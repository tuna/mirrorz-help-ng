项目简介

CocoaPods 是一个 Cocoa 和 Cocoa Touch 框架的依赖管理器，具体原理和 Homebrew 有点类似，都是从 GitHub 下载索引，然后根据索引下载依赖的源代码。

使用方法

对于旧版的 CocoaPods 可以使用如下方法使用镜像：

pod repo remove master
pod repo add master {{endpoint}}/Specs.git
pod repo update



新版的 CocoaPods 不允许用pod repo add直接添加 master 库了，但是依然可以：


cd ~/.cocoapods/repos
pod repo remove master
git clone {{endpoint}}/Specs.git master



最后进入自己的工程，在自己工程的podFile第一行加上：


source '{{endpoint}}/Specs.git'
