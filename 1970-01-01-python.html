Python 官方仓库

{% endraw %}
{% raw %}

此目录镜像了 Python 官方仓库（https://www.python.org/ftp/python/）中的内容，包括源代码和适用于部分系统、架构的预编译二进制文件。

注意：

  • 所有的预发布版本（即版本号中含有 a|b|rc 的版本）均被排除，如有需要，请前往官方仓库下载。
  • 如果您使用的是 Linux 发行版，则通常应当使用发行版自带的软件包管理器安装 Python，而不是直接从本仓库下载和安装二进制文件。
  • 如果您需要 Anaconda，则请参考 Anaconda 软件仓库使用帮助（如果本站提供此镜像）。
